# CODEFEST - Festival Management System

Repository created for the Structured Programming course at CESAE Digital.

This project is a console-based festival management system developed in Java, designed to simulate the administration and attendee experience of a music festival called CODEFEST.

The application uses CSV and TXT file handling to manage and analyze festival data such as tickets, attendees, concerts, stages, quizzes, and revenue reports.

## Technologies Used

* Java
* CSV File Processing
* Structured Programming
* Object-Oriented Organization
* Git & GitHub

## Features

* Admin authentication system
* Ticket sales and revenue analysis
* Attendee search and management
* Festival line-up visualization
* Concert and stage information
* Interactive music quiz
* File reading and matrix manipulation

Developed as part of the practical evaluation project for the Software Developer program at CESAE Digital.
